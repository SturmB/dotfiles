# Keypirinha

Copy all files except `heidisql.ini`, `PackageControl.ini`, and `updater.ini` to `/mnt/c/Keypirinha/portable/Profile/User`.

`PackageControl.ini` is probably only useful to see what packages should be installed. I'm not sure it lists all of them, though.

`heidisql.ini` and  `updater.ini` are zero-byte files so there's no need to do anything with them for now.
